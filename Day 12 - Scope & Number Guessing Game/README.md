## Project Goal
Build a `Number Guessing Game` that:
- generates a random number between 1 and 100
- lets the user choose difficulty (easy/hard)
- gives the user a limited number of attempts
- provides feedback (too high/too low)
- demonstrates proper use of `scope` and `global` variables
