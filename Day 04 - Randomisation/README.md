# Day 04 - Randomisation

## What I Learned

- How to use the `random` module  
- Generating random integers using `random.randint()`  
- Selecting random elements using `random.choice()`  
- Working with Python lists  
- Indexing a list using positions  
- Using negative indexing (`-1`)  
- Understanding and fixing `IndexError` (out of range)  
- Input validation (checking if user input is between 0–2)  
- Using lists to map choices (index → value)
